# Dashboard: About

`/dash/about` is the credits page. Always public; no role required.

---

## What it shows

- Warden version
- Apache License 2.0 link
- Links to the source repo and issue tracker
- Bundled third-party libraries with license attribution (full list in `THIRD_PARTY_LICENSES.md`)
- Optional donation CTA configurable on the Landing tab

This is the page to point someone at if they want to know "what is this project, where can I report a bug, who made it".
